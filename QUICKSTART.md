# 快速启动指南

## 快速开始

### 1. 安装依赖

```bash
cd petraq-admin
npm install
```

### 2. 启动开发服务器

```bash
npm run dev
```

应用将自动在浏览器中打开 `http://localhost:3000`

### 3. 构建生产版本

```bash
npm run build
```

构建完成后，可以在 `dist` 目录找到生产版本的文件。

## 功能说明

### 系统概览
- 查看系统整体统计信息
- 查看同步统计和成功率
- 查看用户动物分布图表
- 查看最近用户和同步记录

### 用户管理
- 点击"添加用户"按钮创建新用户
- 设置用户角色（管理员、普通用户、查看者）
- 设置用户状态（活跃、禁用、待激活）
- 分配用户到组织
- 点击"编辑"按钮修改用户信息
- 点击"删除"按钮删除用户（会同时删除该用户下的所有动物）

### 用户动物管理
- 选择用户查看该用户下的所有动物
- 点击"添加动物"按钮为用户添加动物
- 点击"编辑"按钮修改动物信息
- 点击"删除"按钮删除动物
- 点击"同步数据"按钮手动触发数据同步

### 数据同步管理
- 查看所有同步记录
- 按用户筛选同步记录
- 点击"触发同步"按钮手动启动同步任务
- 选择同步类型（全量、增量、手动）
- 选择同步方向（上传、下载）
- 查看同步状态、耗时、错误信息
- 查看同步统计和成功率

### 组织管理
- 点击"添加组织"按钮创建新组织
- 查看组织统计信息
- 编辑和删除组织

### 系统设置
- 配置同步设置
- 配置通知设置
- 数据导入导出

## 注意事项

1. 当前版本使用模拟数据，所有数据存储在内存中
2. 刷新页面会重置数据
3. 同步任务会模拟2秒的处理时间
4. 同步成功率约为90%（模拟）

## 常见问题

### Q: 页面无法加载？
A: 确保已安装所有依赖：`npm install`

### Q: 端口被占用？
A: 修改 `vite.config.ts` 中的 `port` 配置

### Q: 构建失败？
A: 检查 Node.js 版本是否 >= 16.0.0

## 与 iOS App 的关系

这个管理平台是 **SaaS 管理后台**，用于：
- 管理使用 iOS App 的用户
- 管理用户下的动物数据
- 管理 App 和服务器之间的数据同步
- 系统级的管理和配置

**不是** iOS App 功能的直接复制，而是从管理者的角度提供系统管理功能。
